Расширений файлов в Linux


Архивы и сжатие
  .tar - Архив без сжатия (Tape Archive)
  .tar.gz / .tgz - Сжатый архив gzip
  .tar.bz2 - Сжатый архив bzip2
  .tar.xz - Сжатый архив xz
  .gz - Сжатый файл gzip
  .bz2 - Сжатый файл bzip2
  .xz - Сжатый файл xz
  .zip - Архив ZIP
  .7z - Архив 7-Zip
  .rar - Архив RAR

Исполняемые файлы и скрипты
  .sh - Bash-скрипт
  .bin - Двоичный исполняемый файл
  .run - Установщик (часто самораспаковывающийся)
  .elf - Исполняемый бинарный файл ELF
  (без расширения) - Часто исполняемые файлы не имеют расширения

Исходный код и конфигурация
  .c, .cpp, .h - Исходники на C/C++
  .py - Скрипты Python
  .pl - Скрипты Perl
  .rb - Скрипты Ruby
  .java - Исходники Java
  .js, .ts - JavaScript и TypeScript
  .go - Исходники Go
  .conf - Конфигурационные файлы
  .ini - Конфигурационные файлы INI
  .yaml / .yml - Файлы YAML
  .json - Файлы JSON
  .xml - Файлы XML

Пакеты и установки
  .deb - Установочный пакет Debian/Ubuntu
  .rpm - Установочный пакет Red Hat/Fedora
  .AppImage - Портативное приложение
  .snap - Snap-пакет
  .flatpak - Flatpak-пакет

Документы и текст
  .txt - Текстовый файл
  .md - Markdown-документ
  .log - Файл логов
  .pdf - Документ PDF
  .odt - Документ LibreOffice Writer
  .csv - Таблица CSV

Мультимедиа
  .jpg, .png, .gif, .svg - Изображения
  .mp3, .ogg, .wav, .flac - Аудио
  .mp4, .mkv, .webm, .avi - Видео
